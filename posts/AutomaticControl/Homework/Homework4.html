1. E4.14

先算Open Loop Transfer Function：

Gc=Ks(s+K1)

所以閉路trnadfer Function就是：

H=Gc1+Gc=Ks2+K1s+K

要計算對K1的sensitivity，這裡用一個小技巧：先對H取ln()，然後再微分。這時候會得到：

ln(H)=ln(K)−ln(s2+K1s+K)

dln(H)dK1=dHdK11H=−ss2+K1s+K

因此sensitivity就是：

dln(H)dK1⋅K1=−sK1s2+K1s+K

這就是(a)小題的答案。接著要問如何最小話Td的影響。這時候就要求Y的形式。把Td移動到K前面，並求出Y：

Y=RH+TdKH=KRs2+K1s+K+Tds2+K1s+K

所以，如果讓K變得很大，那麼Td的影響就會相對來說很小了。

2. P4.2

首先把數字帶進去G(s)裡面：

G(s)=9s2+1.2s+9

然後求開路transfer function，乘一個Ka：

Gc=9Kas2+1.2s+9

。接著求Sensitivity。一樣是取ln後用看的：

SGcKa=1

SGcK1=0

會發現前者是1(其實根本不用算，因為開路嘛)，後者是0(因為根本不會影響到)。接著看看Y會長怎樣：

Y(s)=9Kaθds2+1.2s+9+9Tds2+1.2s+9

接著，求閉路transfer function：

T(s)=KaG1+K1KaG=9Kas2+1.2s+(9+K1Ka)

然後算sensitivity，所以用剛剛取ln的招式：

STKa=dln(T)dKa⋅Ka=1Ka−K1s2+1.2s+(9+K1Ka)

STK1=dln(T)dK1⋅Ka=−K1s2+1.2s+(9+K1Ka)

因為取了ln，所以微分根本就是用看的就算出來了。覺得有點開心。接著要消除Td，那就求出Y(s)：

Y=H(θd+TdKa)=9Kaθds2+1.2s+(9+K1Ka)+9Tds2+1.2s+(9+K1Ka)

4. P4.12

顆顆，又是差不多的題目。來寫個code解他：
首先先宣告需要的變數：

syms s;
syms K1;
syms K2;

然後這是題目給定的兩個開路Transfer Function

Gc_a = K1 / (s + 4);
Gc_b = K2 / (s - 1);

由此可以求出兩個系統的Transfer Function。對於第一個而言，Gca,Gcb先串再回授，所以：

T1 = (Gc_a * Gc_b)/(1 + Gc_a * Gc_b * 6);

對於第二個系統，是兩個先回授再串連起來：

T2 = ((Gc_a)/(1 + Gc_a * (-2))) * ... 
     ((Gc_b)/(1 + Gc_b * (2)));

求出來之後，叫電腦把它們化簡一下：

T1 = simplify(expand(T1));
T2 = simplify(simplify(T2));

然後顯示出來：

disp('T1');
pretty(T1);
disp('T2');
pretty(T2);

這時候matlab會輸出：

這就是第一題需要的答案了。然後題目要求兩系統各自對K1,K2的sensitivity，這裡一樣使用剛剛取ln的招式，先取ln，再微分，再乘一個變數。對於系統1：

S11 = diff(log(T1), K1, 1) * K1;
S12 = diff(log(T1), K2, 1) * K2;

對於系統2：

S21 = diff(log(T2), K1, 1) * K1;
S22 = diff(log(T2), K2, 1) * K2;

然後化簡系統1分別對K1,K2的sensitivity：

S11 = simplify(expand(S11));
S12 = simplify(expand(S12));

接著，題目要求把將K1,K2代入1。這裡使用subs()函數：

ans1 = subs(S11, K1, 1);
ans1 = subs(ans1, K2, 1);
ans2 = subs(S12, K1, 1);
ans2 = subs(ans2, K2, 1);

sub()函數第一個參數是「要處理的表示式」，第二個參數是「要把這個表示式中哪一個變數帶掉」，大三個參數是「要帶什麼值進去」。以第一條為例：

ans1 = subs(S11, K1, 1);

意思是ans1等於「把S11中的K1帶成1」。分別帶完之後顯示出來：

disp('S11, S12');
pretty(ans1);
pretty(ans2);

而這時matlab會輸出：

對系統2的兩個sensitivity做同樣的代入：

S21 = simplify(expand(S21));
S22 = simplify(expand(S22));
ans1 = subs(S21, K1, 1);
ans1 = subs(ans1, K2, 1);
ans2 = subs(S22, K1, 1);
ans2 = subs(ans2, K2, 1);

並且顯示出來：

disp('S21, S22');
pretty(ans1);
pretty(ans2);

matlab輸出為：

這裡可以得到以下觀察：

  • 很悲劇的，系統2對K1的senstivity比1還要大…誰叫他要拉正回授。
  • 不過觀察系統2的Transfer Function可以發現他的pole很好調。然後發現這根本跟Root Locus的概念有87%像。
  • 而對於系統1，對K1, K2兩者的sensitivity都一樣。聽起來也很合理，因為K1, K2在Transfer Function中是對稱的，如果要他們交換位置之後還是一樣，那兩個微出來的結果也應該要一樣。
