( c )這下子要把他從原點向右偏移0.002，做一條直線上去，找焦點。真的是一件有點痛苦的事。所以我決定寫code處理它。如下：

#include<cstdio>
#include<cmath>

struct point{
	double x;
	double y;
	point(double _x, double _y):x(_x), y(_y){}
};
int main()
{
	/* m是楊氏係數，也是直線的斜率，所以叫m */
	double m = 1.8e11;	
	double x, y;

	/* 只是宣告兩個點。裡面數字沒意義 */
	point P0(0, 0);
	point P1(0, 0);

	/* 把直線L: m(x-0.002)=y, 表成(a1)x+(b1)y=c1 */
	double a1 = m;
	double b1 = -1;
	double c1 = 0.002 * m;

	/* 讀取原點 */
	scanf("%lf%lf", &P0.x, &P0.y);
	
	/* 暴力算每一個線段延伸出的直線，與L的交點 */
	/* 如果該點在線段內，表示是與應力-應變曲線的交點 */
	while(scanf("%lf%lf", &P1.x, &P1.y) != EOF){
	
		/* 用法向量求直線,表成(a2)x+(b2)y=c2 */		
		double a2 = P0.x - P1.x;
		double b2 = -1 * (P0.y - P1.y);
		double c2 = a2 * P0.x + b2 * P0.y;

		/* 克拉瑪公式 */
		double D = a1 * b2 - a2 * b1;
		double D_x = c1 * b2 - c2 * b1;
		double D_y = a1 * c2 - a2 * c1;
		x = D_x / D;
		y = D_y / D;

		/* 如果交點在線段中間，就顯示出來 */
		if(x-P0.x > 0 && x-P0.x < P1.x-P0.x){
			printf("%lf %lf\n", x, y);
		}

		/* 把現在線段的終點，變成下一個線段的起點 */
		P0.x = P1.x;
		P0.y = P1.y;
	}//回到while迴圈頭
	return 0;
}

( a )
如果鐵片尺寸夠厚，而且可以抽幾片來測試，那麼Brinell與Rockwell均可。如果鐵片很薄，那麼Vickers比較妥當。如果是超大，塞不下測試機的超大鐵片，那麼用Shore。

( b )
Brinell測試比較妥當，因為鑄鐵是不均勻的材料，所以需要比較大的壓痕，才有代表性。剛好Brinell硬度測試的壓痕最大，其他的都只能測局部硬度(如果不考慮Shore的話)。

( c )
Vickers可以做到很小的尺度，所以就選Vickers吧～Brinell(錶大概會爆掉)、Roclwell(痕跡太明顯)、Shore(沒有足夠大的表面給他彈)都不太適合。

一樣是寫code算：

#include<stdio.h>
#include<math.h>
int main()
{
    double dat[10];
    double avg = 0;
    double dev = 0;

    /* 把10筆資料讀到dat陣列中，順便一路累計算總和 */
    for(int i = 0; i < 10; i++){
	    scanf("%lf", &dat[i]);
	    avg += dat[i];
    }
	
    /* 算平均 */
    avg /= 10;
    printf("average = %lf\n", avg);

    /* 算每一筆資料對 標準差平方 的貢獻 */
    for(int i = 0; i < 10; i++){
	    dev += (dat[i] - avg)*(dat[i] - avg)/9.0;
    }
	
    /* 開根號之後就算出標準差惹 */
    dev = sqrt(dev);
    printf("dev = %lf\n", dev);

    return 0;
}

算出來之後得到

average = 85.230000  (平均)
dev = 2.393533 (標準差)

材料用不同的速度拉伸下時，Toughness有可能不同。舉例來說，如果用力敲打瀝青(strain rate大概在103/s103/s左右)，瀝青可能瞬間就碎裂;但是如果以Tensile Test的緩慢速度拉伸瀝青(strain rate大概在10−3/s10−3/s)，那麼瀝青可以被拉伸的很長，也不會立刻碎裂。這就是不同Strain Rate對材料性質的影響。
