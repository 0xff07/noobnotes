Miller Indice

Miller Indeces for CUBICS

結晶之格子面(lattice plane)

結晶之格子連線方向(lattice direction)

以上描述方式均由解析幾何而來

$$\frac {x} {a} + \frac {y} {b} + \frac {z} {c} = 1$$ (a plane in x-y-z plane)

$$h = \frac {1} {a}, \space k = \frac {1} {b}, \space l = \frac {1} {c}$$

\(\Rightarrow\)i.e. use (h k l) (並且把3個數字 約分成整數(可以有負數，上面會有一槓)) to describe a plane

  • 3個數字中間沒有逗號。
  • 平行的面會有一樣的構造，所以視為一樣，Miller Indices也一樣

(看課本例子)

$$\left \{100\right \}表示{(100), (010), (001), (\bar{0}10), (0\bar{1}0), (00\bar{1})}這些平面的集合，叫做「family\space\space of\space plane」$$

$$<100>表示{[100], [010], [001], [\bar{0}10], [0\bar{1}0], [00\bar{1}]}這些方向的集合，叫做「family\space\space of\space direction」$$

Miller - Bravais Indices

Miller Indices for HCP

把Miller Indeces推廣到HCP上。方法是 多新增一個維度

這是一個HCP構造：

在底面設3個軸x-y-z，高度另外設一個軸w。這時候會有一個問題：空間中自由度只有3，四個座標會導致表法不唯一，該如何處理？

解：新增一個相依條件 i + j + k = 0

通常是i, j, k先隨便找兩個，然後再用i + j + k = 0求出剩下那一個。

另外，因為可能有很多組表達方法，所以原則是要讓(hkil)跟[hkil]數字比例要盡量簡單。

相律

  • 由熱力學導出，是支配物質態(material state)平衡關係的重要法則：

    $$F = n - r + 2$$

    $$F : degree \space of \space freedom$$

    $$n : no. \space of \space components$$

    $$r : no. \space of \space phases$$

    後面那個2，是表示「溫度」與「壓力」兩個變量。

  • 舉例來說，我想知道水三向平衡時，那會是個「三向點」、「三向線」，還是「三向面」？只要帶入相律：

    $$F = 1 - 3 + 2 = 0$$

    因為F = 0，所以是個0維的東西，也就是三像共存時，在T - P圖上是一個點。

    那水有沒有可能四相共存？一樣可以用相律驗證。假設有四項共存，則自由度為：

    $$F = 1 - 4 + 2 = -1$$

    很顯然自由度不能比0小。所以相律預測不可能存在。

  • 再舉例如純鐵的相圖：

    可以發現他還是只有3相點。

  • 再舉例來說，如果你想要固定壓力，那麼只又把相律改成：

    $$F = n - r + 1$$

    因為溫度與壓力有一個自由度被強制固定了，所以少一個自由度。

    所以一個Cu - Zn系統，在定壓下：

    • 2相平衡時，$$F = 2 - 2 + 1 = 1$$是一條線。

    • 3相平衡時，$$F = 2 - 3 + 1 = 0$$是一個點。是一個點的狀況，叫做「invariant condition」

全率固溶型：

    這個圖上有一些特別的線可以注意一下。首先，隨便畫一條線，這裡是LS。那麼這就是一條「繫線」

  • 繫線(tie line)：LS連線，一條向圖上的橫線。繫線可以知道當下凝固出來的金屬中，AB比例是多少。只要你把現左右延伸到固相線與液相線兩端就可以了。 在這個例子中，析出的固體中，A 的比例佔\(W_{l}\)，B的比例佔\(W_{s}\)

  • 當繫線往下移動時，可以由交點發現析出固體中，A的比例會漸少，B的比例會漸多。

  • 雖然理論上來說，每一刻析出的固體，AB比例應該都會不一樣>但是如果凝固速度夠慢，不同層之間會彼此擴散，所以最後就可以形成大致均勻的晶體。這種現象就叫「偏析(segregation)」
